Description
This feature allows the CRA IT Security Team to create detailed summaries of phishing-related activity within the organization. This includes information such as the number of phishing emails reported by employees, the detection accuracy of the phishing prevention system ans patterns in phishing attempts over time. Users can select specific date ranges and report type.
